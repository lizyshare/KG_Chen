# 启动指南
- 进入font_end文件夹

  执行以下命令：
  `npm install`
  `npm run dev`

- 进入back_end文件夹
  新建环境 `pip install -r requirements.txt`

  执行以下命令：

  `python app.py`
  
- 最后访问地址  http://localhost:5173/ 进入问答系统。
